Privacy Statement for Listen Carefully Browser Extension

Last Update: April 4, 2026
Extension version: 1.1.2

At Power to Logic, we take your privacy seriously. This Privacy Statement outlines how we collect, use, and protect your personal information when you use our browser extension.

We want to assure you that Listen Carefully is designed to prioritize your data security and privacy. Listen Carefully does not collect, store, or transmit any user data to external servers. We do not use telemetry or network connectivity for data sharing.

Data Collection and Usage

Listen Carefully is a browser extension that operates entirely within your browser and, optionally, with a locally hosted TTS server on your machine. We do not collect any personal data from you, and we do not have access to or store any data on external servers or networks. The extension only stores your preferences locally on your device using the browser's built-in storage. Listen Carefully does not use telemetry or network connectivity to send or receive data externally.

Text-to-Speech Processing

Listen Carefully supports two TTS backends:

  • Browser (Web Speech API): The default backend. Uses the speech synthesis built into your browser and operating system. No page content is sent to the creators of this extension or any third parties affiliated with us. Please note that some browser or operating system voices may connect to external services (e.g., cloud-based speech synthesis provided by your browser or OS vendor) to generate audio. This behavior is controlled by your browser or operating system and is outside the scope of this extension. If you prefer fully offline processing, select a voice that is installed locally on your device.
  • Kokoro (Local API): An optional advanced feature for users who want high-quality natural voices while keeping all TTS communication fully on their local machine. When enabled, page text for the current sentence is sent to a Kokoro TTS server running on your own device (localhost). This data never leaves your machine — the extension enforces that the Kokoro endpoint must be a localhost address (localhost, 127.0.0.1, or [::1]). This feature requires you to install and run the Kokoro server separately and to grant an optional localhost permission, which is requested only when you enable the Kokoro backend in settings. No data is sent to external servers.

Permissions

Listen Carefully requests certain browser permissions for its core functionalities. These permissions are used exclusively for the purpose of the extension's features and do not involve the collection or transmission of personal data.

  • Storage: Listen Carefully uses the storage permission to save your preferences such as voice selection, playback speed, highlight colors, and focus mode settings. This data remains on your device and is not shared with external servers or third parties.
  • Active Tab: This permission allows Listen Carefully to access the content of the current page in order to read text aloud and apply word-level highlighting. Content scripts are injected into web pages (http://, https://) and optionally local files (file://) when "Allow access to file URLs" is enabled in your browser's extension settings. Page content is processed locally and is never transmitted.
  • Scripting: This permission is used to inject the text-to-speech and highlighting functionality into web pages. No scripts are loaded from external sources.
  • Context Menus: This permission enables the "Read from here" right-click menu option for convenient access to the reader.
  • Localhost Access (optional): This permission is only requested when you enable the Kokoro TTS backend in settings. It allows the extension to communicate with a TTS server running on your own machine. The permission is limited to localhost and 127.0.0.1 addresses and cannot be used to contact external servers. If you do not use the Kokoro feature, this permission is never requested or granted.
